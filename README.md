# Dharaneesh D - Portfolio

This is a simple, responsive, and modern portfolio website built with HTML, CSS, and JavaScript.

## Features
-   **Dark Theme**: A professional and comfortable dark mode design.
-   **Responsive**: Looks great on mobile, tablet, and desktop.
-   **Animations**: Smooth scrolling and scroll-reveal animations.
-   **Sections**: About, Skills, Projects, Experience, Education, Contact.

## How to Edit
1.  **Content**: Open `index.html` and modify the text within the tags.
2.  **Styles**: Open `style.css` to change colors or layout. The colors are defined in the `:root` variables at the top.
3.  **Images**: If you want to add images (e.g., for projects), replace the icons or add `<img>` tags in the HTML.

## Run Locally
You can simply open `index.html` in your web browser.
For a better development experience, you can use a live server extension in VS Code.
